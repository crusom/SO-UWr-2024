### Przeglądanie plików
Dla wygody przeglądania rozwiązań dołączone są pliki html oraz skrypty je generujące.
Aby przeglądać writeupy wystarczy wejść w odpowiedni katalog z numerem listy i odpalić dowolony webserver np. pythonowy moduł http.server.
```sh
$ cd 3/
$ python -m http.server -d html/
Serving HTTP on 0.0.0.0 port 8000 (http://0.0.0.0:8000/) ...
```
### Programy i biblioteka
Do list dołączone są programy, które trzeba było napisać/uzupełnić, znajdują się one w podkatalogach so21_lista_X.
Nie dołączyłem biblioteki libcsapp potrzebnej do kompilacji.
### Poprawność programów
Pliki w C, w tym repozytorium, są takimi jakimi je zostawiłem po prezentacji na zajęciach.
Nie gwarantuję, że są w pełni poprawne ani że w pełni odpowiadają poleceniu (w trakcie zajęć były często eksperymenty z programami).
### Poprawność writeupów
Choć starałem się, aby moje odpowiedzi były rozwinięte, to czasami ich nie zapisywałem, bo łatwiej mi było o problemie opowiedzieć z głowy na zajęciach, dlatego rozwiązania zdecydowanie nie są pełne.
Rozwiązania na pewno nie są w pełni poprawne, proszę z nich korzystać z rozwagą.
W szczególności nie mam pewności co do moich odpowiedzi do zadań z synchronizacji, dlatego proszę podchodzić do nich z przymróżeniem oka.
